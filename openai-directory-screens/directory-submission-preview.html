Screenshots for OpenAI Directory

Requirements & Content Rules

  • Screenshots: Minimum 1, maximum 4, all in 2x (retina) quality.
  • Formats: PNG or JPG.
  • Dimensions: Width 706px, height 400–860px (860px recommended).
  • Figma template: Use the public Figma template to export compliant images easily.
  • Content rules:
    • Show only the app widget UI (no ChatGPT UI).
    • Do not include user prompts or model responses in the image.
    • Avoid embedded text; localization happens via fields.

Ignoring these content rules may lead to app rejection.
